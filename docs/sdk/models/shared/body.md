# Body


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `{ [k: string]: any }`

```typescript
const value: { [k: string]: any } = {
  "key": "<value>",
  "key1": "<value>",
  "key2": "<value>",
};
```

### `number[]`

```typescript
const value: number[] = [
  339048,
  133621,
  943799,
];
```
