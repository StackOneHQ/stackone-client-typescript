# AnswerSourceValue

The source value of the answer type.


## Supported Types

### `string`

```typescript
const value: string = "Short Text";
```

### `number`

```typescript
const value: number = Short Text;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.Answer4`

```typescript
const value: shared.Answer4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "Short Text",
];
```
