# ContentBlockSourceValue

The source value of the status.


## Supported Types

### `string`

```typescript
const value: string = "active";
```

### `number`

```typescript
const value: number = 1284.03;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.ContentBlock4`

```typescript
const value: shared.ContentBlock4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "<value 1>",
  "<value 2>",
  "<value 3>",
];
```
