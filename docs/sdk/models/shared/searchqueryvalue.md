# SearchQueryValue

The value to search for


## Supported Types

### `string`

```typescript
const value: string = "john";
```

### `number`

```typescript
const value: number = NaN;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.SearchQuery4`

```typescript
const value: shared.SearchQuery4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "john",
];
```
