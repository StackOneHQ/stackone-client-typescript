# CreateAnswerSourceValue

The source value of the answer type.


## Supported Types

### `string`

```typescript
const value: string = "Short Text";
```

### `number`

```typescript
const value: number = 1284.03;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.CreateAnswer4`

```typescript
const value: shared.CreateAnswer4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "<value 1>",
  "<value 2>",
  "<value 3>",
];
```
