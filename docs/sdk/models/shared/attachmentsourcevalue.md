# AttachmentSourceValue

The source value of the content type.


## Supported Types

### `string`

```typescript
const value: string = "Text";
```

### `number`

```typescript
const value: number = NaN;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.Attachment4`

```typescript
const value: shared.Attachment4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "Text",
];
```
