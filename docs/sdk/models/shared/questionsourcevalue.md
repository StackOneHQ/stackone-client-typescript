# QuestionSourceValue

The source value of the questions type.


## Supported Types

### `string`

```typescript
const value: string = "ShortText";
```

### `number`

```typescript
const value: number = NaN;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.Question4`

```typescript
const value: shared.Question4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "ShortText",
];
```
