# QuestionSourceValue

The source value of the questions type.


## Supported Types

### `string`

```typescript
const value: string = "ShortText";
```

### `number`

```typescript
const value: number = 1284.03;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.Question4`

```typescript
const value: shared.Question4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "<value 1>",
  "<value 2>",
  "<value 3>",
];
```
