# ContentBlockSchemasSourceValue

The source value of the type.


## Supported Types

### `string`

```typescript
const value: string = "text";
```

### `number`

```typescript
const value: number = text;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.ContentBlockSchemas4`

```typescript
const value: shared.ContentBlockSchemas4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "text",
];
```
