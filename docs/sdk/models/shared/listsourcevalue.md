# ListSourceValue

The source value of the list type.


## Supported Types

### `string`

```typescript
const value: string = "Contacts";
```

### `number`

```typescript
const value: number = 1284.03;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.List4`

```typescript
const value: shared.List4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "<value 1>",
  "<value 2>",
  "<value 3>",
];
```
