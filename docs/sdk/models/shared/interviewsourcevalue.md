# InterviewSourceValue

The source value of the interview status.


## Supported Types

### `string`

```typescript
const value: string = "Unscheduled";
```

### `number`

```typescript
const value: number = 1284.03;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.Interview4`

```typescript
const value: shared.Interview4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "<value 1>",
  "<value 2>",
  "<value 3>",
];
```
