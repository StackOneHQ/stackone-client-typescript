# InterviewSourceValue

The source value of the interview status.


## Supported Types

### `string`

```typescript
const value: string = "Unscheduled";
```

### `number`

```typescript
const value: number = Unscheduled;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.Interview4`

```typescript
const value: shared.Interview4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "Unscheduled",
];
```
