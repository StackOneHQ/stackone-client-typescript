# CampaignSourceValue

The source value of the schedule type.


## Supported Types

### `string`

```typescript
const value: string = "Immediate";
```

### `number`

```typescript
const value: number = Immediate;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.Campaign4`

```typescript
const value: shared.Campaign4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "Immediate",
];
```
