# CampaignSourceValue

The source value of the schedule type.

```typescript
const value: string = "Immediate";
```

### `number`

```typescript
const value: number = 1284.03;
```

```typescript

```

### `shared.Campaign4`

```typescript
const value: shared.Campaign4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "<value 1>",

  "<value 3>",
];
```
