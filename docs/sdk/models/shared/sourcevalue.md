# SourceValue


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 4103.02;
```

### `boolean`

```typescript
const value: boolean = false;
```

### `shared.Four`

```typescript
const value: shared.Four = {};
```

### `any[]`

```typescript
const value: any[] = [
  "<value>",
];
```
