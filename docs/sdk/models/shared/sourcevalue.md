# SourceValue

The source value of the Channels.


## Supported Types

### `string`

```typescript
const value: string = "SMS";
```

### `number`

```typescript
const value: number = SMS;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.Four`

```typescript
const value: shared.Four = {};
```

### `any[]`

```typescript
const value: any[] = [
  "SMS",
];
```
