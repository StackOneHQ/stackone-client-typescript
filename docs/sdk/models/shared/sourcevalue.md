# SourceValue

The source value of the Channels.


## Supported Types

### `string`

```typescript
const value: string = "SMS";
```

### `number`

```typescript
const value: number = 1284.03;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.Four`

```typescript
const value: shared.Four = {};
```

### `any[]`

```typescript
const value: any[] = [
  "<value 1>",
  "<value 2>",
  "<value 3>",
];
```
