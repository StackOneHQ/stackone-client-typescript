# CustomFieldOptionValue

The human readable value of the option


## Supported Types

### `string`

```typescript
const value: string = "Not Started";
```

### `number`

```typescript
const value: number = NaN;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.CustomFieldOption4`

```typescript
const value: shared.CustomFieldOption4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "Not Started",
];
```
