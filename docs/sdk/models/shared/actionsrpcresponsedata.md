# ActionsRpcResponseData

The response data from the action RPC call


## Supported Types

### `{ [k: string]: any }`

```typescript
const value: { [k: string]: any } = {
  "key": "<value>",
  "key1": "<value>",
  "key2": "<value>",
};
```

### `{ [k: string]: any }[]`

```typescript
const value: { [k: string]: any }[] = [];
```
