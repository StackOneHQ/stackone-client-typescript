# AccountAddressSchemasSourceValue

The source value of the ISO 3166-1 alpha-2 code of the country.

```typescript

```

### `number`

```typescript
const value: number = 1284.03;
```

```typescript

```

```typescript
const value: shared.AccountAddressSchemas4 = {};
```

