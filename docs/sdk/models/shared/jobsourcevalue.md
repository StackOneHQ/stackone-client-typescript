# JobSourceValue

The source value of the job status.


## Supported Types

### `string`

```typescript
const value: string = "Published";
```

### `number`

```typescript
const value: number = Published;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.Job4`

```typescript
const value: shared.Job4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "Published",
];
```
