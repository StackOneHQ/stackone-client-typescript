# OfferSourceValue

The source value of the offer status.


## Supported Types

### `string`

```typescript
const value: string = "Pending";
```

### `number`

```typescript
const value: number = Pending;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `shared.Offer4`

```typescript
const value: shared.Offer4 = {};
```

### `any[]`

```typescript
const value: any[] = [
  "Pending",
];
```
